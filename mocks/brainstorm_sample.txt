VÒNG 1: Ý TƯỞNG TỰ DO (30 phút)

1. Real-time transcription trong cuộc họp
   - Hiển thị transcript ngay khi người nói
   - Hỗ trợ nhiều ngôn ngữ đồng thời

2. AI Meeting Assistant với voice
   - User có thể hỏi AI bằng giọng nói
   - AI trả lời bằng voice synthesis

3. Smart action items extraction
   - Tự động phát hiện task và assign người
   - Tích hợp với Jira, Trello

4. Meeting analytics dashboard
   - Thống kê thời gian nói của mỗi người
   - Phân tích sentiment và engagement

5. Auto-generate meeting slides
   - Tạo PowerPoint từ transcript
   - Visual summary với charts

6. Multi-language support
   - Dịch real-time sang nhiều ngôn ngữ
   - Hỗ trợ 50+ ngôn ngữ

7. Video meeting integration
   - Tích hợp với Zoom, Google Meet
   - Record và analyze video

8. AI-powered search
   - Tìm kiếm trong tất cả meetings
   - Semantic search với RAG

VÒNG 3: PHÁT TRIỂN Ý TƯỞNG (40 phút)

Ý tưởng được chọn: Real-time transcription

Chi tiết:
- Technology: WebSocket + Whisper API
- UI: Live transcript panel bên cạnh video
- Features:
  + Auto-scroll theo người nói
  + Highlight keywords
  + Speaker diarization
  + Export transcript
  
Challenges:
- Latency phải < 2 giây
- Cost optimization cho API calls
- Accuracy với nhiều accent

Solutions:
- Use streaming API
- Implement caching
- Fine-tune model cho tiếng Việt
